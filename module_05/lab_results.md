# Результаты выполнения лабораторной работы — Модуль 5

## Примечание

Лабораторная работа модуля 5 посвящена операциям DML (INSERT, UPDATE, DELETE, MERGE, UPSERT). Результаты выполнения зависят от предварительного создания practice-таблиц с помощью скрипта `scripts/create_practice_tables.sql`.

DAX является языком запросов только для чтения и не поддерживает операции DML. Поэтому задания данного модуля выполняются исключительно на SQL (PostgreSQL).

---
