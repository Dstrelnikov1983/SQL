## Задание 1. Скалярная функция — расчёт OEE

**Тестовые вызовы:**

```
calc_oee(10, 12, 80, 100) = 66.7
calc_oee(12, 12, 100, 100) = 100.0
calc_oee(8, 12, 0, 100) = 0.0
```

> Формула: OEE = (operating / planned) * (actual / target) * 100. При нулевой добыче OEE = 0, при нулевом плане — NULL.
